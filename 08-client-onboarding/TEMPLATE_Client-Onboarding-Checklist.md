> This checklist provides a structured process for onboarding a new D2C client. Following these steps will ensure that you gather all the necessary information and assets to set up their email marketing program for success.

# Client Onboarding Checklist: [CLIENT NAME]

**Date:** `YYYY-MM-DD`

---

## Phase 1: Pre-Kickoff

*   [ ] **Send Welcome Packet:** Send the client a welcome email that includes:
    *   An overview of the onboarding process.
    *   A link to schedule the kickoff meeting.
    *   A request for access to key platforms.
*   [ ] **Gain Access to Platforms:**
    *   [ ] Shopify (with necessary permissions).
    *   [ ] Klaviyo (or create a new account for them).
    *   [ ] Google Analytics.
    *   [ ] Any other relevant marketing or data platforms.
*   [ ] **Send Initial Questionnaires:** Send the client the following documents to fill out before the kickoff meeting:
    *   [ ] `TEMPLATE_Brand-Research-and-Discovery.md`

---

## Phase 2: Kickoff Meeting

*   [ ] **Host Kickoff Meeting:** Use the `TEMPLATE_Kickoff-Meeting-Agenda.md` to guide the meeting.
*   [ ] **Review Discovery Document:** Go through the `TEMPLATE_Brand-Research-and-Discovery.md` document with the client to ensure a deep understanding of their brand.
*   [ ] **Define Goals & KPIs:** Solidify the primary goals and Key Performance Indicators (KPIs) for the email marketing program.
*   [ ] **Establish Communication Cadence:** Agree on a schedule for regular check-in meetings and reports.

---

## Phase 3: Strategy & Setup

*   [ ] **Complete Strategy Documents:** Based on the information gathered, complete the following strategy documents:
    *   [ ] `TEMPLATE_Brand-Voice-and-Tone.md`
    *   [ ] `TEMPLATE_Style-Guide.md`
    *   [ ] `TEMPLATE_Customer-Avatar.md` (for 2-3 key segments)
    *   [ ] `TEMPLATE_Segmentation-Strategy.md`
    *   [ ] `TEMPLATE_Merchandising-Strategy.md`
*   [ ] **Present Strategy for Approval:** Share the completed strategy documents with the client for feedback and approval.
*   [ ] **Technical Setup:**
    *   [ ] Configure Klaviyo-Shopify integration.
    *   [ ] Set up DNS records for a dedicated sending domain.
    *   [ ] Install Klaviyo onsite tracking (for Active on Site and Viewed Product metrics).
    *   [ ] Set up the Klaviyo back-in-stock feature on the Shopify theme.

---

## Phase 4: Implementation

*   [ ] **Build Core Segments:** Create the priority segments in Klaviyo as defined in the `TEMPLATE_Segmentation-Strategy.md`.
*   [ ] **Build Core Flows:** Build, test, and set live the foundational email flows:
    *   [ ] Welcome Series
    *   [ ] Abandoned Cart
    *   [ ] Browse Abandonment
    *   [ ] Post-Purchase
*   [ ] **Create Master Email Template:** Design and build a reusable master email template in Klaviyo that aligns with the `TEMPLATE_Style-Guide.md`.
*   [ ] **Plan First Campaign:** Collaborate with the client to plan the first email campaign.

---

## Phase 5: Go-Live & Handoff

*   [ ] **Final Review:** Conduct a final review of all live flows and segments with the client.
*   [ ] **Provide Training:** Provide the client with training on how to use the new system and templates.
*   [ ] **Schedule First Performance Review:** Set a date for the first monthly or quarterly performance review meeting.
